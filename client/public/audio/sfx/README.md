# Audio Placeholders

Place your `.mp3` files here:

- `pin_drop.mp3` — Card pinned to corkboard
- `rope_attach.mp3` — Rope connected between cards
- `ui_click.mp3` — Button click
- `ui_hover.mp3` — Button hover
- `evidence_glow.mp3` — Evidence discovered
- `chat_message.mp3` — Chat message received
- `transition.mp3` — Scene transition

The audio manager loads these from `/audio/sfx/`. Missing files are handled gracefully (no crash).
